# GitHub Rate Limit Prometheus Exporter

A [Prometheus](https://prometheus.io/) exporter which scrapes GitHub API for the rate limits used by PAT/GitHub App.


## History

This repository started as a fork of [kalgurn/github-rate-limits-prometheus-exporter](https://github.com/kalgurn/github-rate-limits-prometheus-exporter)
and has since diverged substantially from upstream. For that reason it is now
maintained as a standalone project.

Huge thanks to [@kalgurn](https://github.com/kalgurn) for the original work
that this exporter is built on.
